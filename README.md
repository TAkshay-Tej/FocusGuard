# 📵 FocusGuard – AI Digital Wellbeing App

FocusGuard is an AI-powered Android application designed to help users reduce phone addiction, improve focus, and build healthier digital habits through intelligent tracking, smart interventions, and behavioral insights.

---

## 🚀 Features

### 📊 App Usage Tracking
- Tracks per-app screen time using Android UsageStatsManager
- Detects binge usage patterns in real time

### 🚫 Smart App Blocking
- Blocks distracting apps using Overlay Permission
- Triggers interventions when usage thresholds are exceeded

### ⏳ Focus Mode (Pomodoro)
- Enables distraction-free work sessions
- Temporarily restricts selected apps during focus periods

### 🧠 AI Behavioral Insights
- Uses Groq LLaMA to analyze 7-day usage patterns
- Generates personalized addiction insights and focus recommendations

### 🔐 Secure Backend
- JWT-based authentication
- Syncs usage data and analytics across devices
